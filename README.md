# 图灵小书架

#### 项目介绍
定时抓取图灵社区官网的首页、最热、推荐和最新等栏目的相关信息进行展示，并且可以下载相关的 PDF 进行查阅...后台免部署，直接将代码引入微信开发者工具会自动连接到服务器进行数据交互和展示...
